This is the guide for all the installs needed to make the application work.

You need to have Node.js installed.
Once you open the project for the first time, do this:
The terminal commands:
    npm install

    chakra libraries:
    npm i @chakra-ui/react @emotion/react @emotion/styled framer-motion
    npm i @chakra-ui/icons

    Carousel component:
    npm i react-slick
    npm install slick-carousel
    npm i react-icons

    npm install react-router-dom@latest
    npm install @tanstack/react-query@4.28
    npm install @tanstack/react-query-devtools@4.28

    npm i axios 

To run tests:
    You need to install webdriver for Chrome

    Terminal command:
        npm test

npm install --save-dev mocha chai
